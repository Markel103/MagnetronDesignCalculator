    Assumptions
    • Analytic sizing model: outputs are first-pass estimates from closed-form relations (Carter/Collins/Clogston). Not a PIC simulation and not a full 3D EM/thermal/mechanical design.
    • Efficiency and impedance inputs: \(\eta\) and \(Z_{dc}\) are treated as operating-point assumptions/overrides; type defaults are heuristic fits and may differ from a specific tube design.
    • Geometry approximations: cavities are approximated with simple rectangular primitives for \(Q\) and thermal area; straps/rising-sun details and end-hats are not explicitly modeled.
    • Thermal estimate: vane-tip temperature uses a simplified 1D conduction model with fixed material properties and a constant coolant temperature; it’s intended as a screening check.
    • Breakdown check: Kilpatrick is used as a classic RF heuristic; it does not guarantee absence of breakdown in the real tube.
    • Magnet system: axial field \(B_z\) is treated as an available operating parameter; the permanent-magnet/electromagnet design, saturation, and field uniformity are not designed here.
    • Sensitivity panel: each perturbation reruns the full model with one input changed by \(\pm 1\%\) while holding \(N_v\) fixed (same sweep row is extracted at the perturbed point). If that \(N_v\) becomes infeasible, the result is shown as N/A.
    • Build plausibility panel: permanent-magnet plausibility uses only the required \(B_z\) magnitude (does not model magnetic circuit geometry); cooling uses the model’s \(T_v\) screening thresholds; anode loss uses \(P_{loss}=P_{dc}-P_{rf}\) (peak) and duty scaling for average loss.
    Core operating-point & geometry formulas used
    • Definitions
      • RF output power: \(P_{rf}\)
      • DC impedance: \(Z_{dc} = V_a/I_a\)
      • Overall efficiency: \(\eta = P_{rf}/P_{dc}\)
      • Circuit efficiency: \(\eta_C\)
      • Electronic efficiency used in Carter’s operating-point model: \(\eta_E = \eta/\eta_C\)
      • Free-space wavelength: \(\lambda = c/f\)
    • DC operating point (Carter §15; implemented in compute_dc_point)
      • \(P_{dc} = P_{rf}/\eta\)
      • \(V_a = \sqrt{P_{dc}\,Z_{dc}}\)
      • \(I_a = P_{dc}/V_a\)
      • Carter Eq. 15.49 (inverse form used in code): \(\eta_E \approx \frac{2r-3}{2r-1}\;\Rightarrow\; r = \frac{3-\eta_E}{2\,(1-\eta_E)}\)
      • Carter Eq. 15.22: \(V_0 = \frac{V_a}{2r-1}\)
      • Hull cut-off voltage: \(V_H = r^2\,V_0\)
      • Operating ratio reported: \(V_a/V_H\)
    • Material / conductor loss primitives
      • Copper skin depth: \(\delta_s = \sqrt{\frac{\rho_{Cu}}{\pi f\,\mu_0}}\)
      • Copper surface resistance: \(R_s = \sqrt{\pi f\,\mu_0\,\rho_{Cu}}\)
    • Synchronous rotation and radii (Carter Eq. 15.22; implemented in sweep_vanes)
      • Number of cavities: \(n = N_v/2\)
      • Synchronous angular frequency: \(\omega_s = 2\pi f / n\)
      • Anode radius: \(r_a = \frac{\sqrt{2\,(e/m)\,V_0}}{\omega_s}\)
      • Modified Slater factor closed form (Carter Eq. 15.154): \(\xi \equiv r_c/r_a = \frac{N_v\,(r-1) - R'\,r}{N_v\,(r-1) + R'\,r}\)
      • Cathode radius: \(r_c = \xi\,r_a\)
    • Magnetic fields (Carter Eq. 15.23)
      • Characteristic field: \(B_0 = \frac{2\,\omega_s}{(e/m)\,(1-\xi^2)}\)
      • Operating axial field: \(B_z = r\,B_0\)
      • Cyclotron ratio (used for heuristic checks): \(\omega_c/\omega_s = (e/m)\,B_z/\omega_s\)
    • Geometry primitives
      • Anode axial length (heuristic): \(L_a = k\,\lambda\), with \(k\approx 0.16\) (CW) or \(k\approx 0.12\) (pulsed)
      • Vane pitch: \(p_t = 2\pi r_a/N_v\)
      • Gap width (fill factor): \(w_{gap} = \mathrm{fill}\cdot p_t\)
      • Vane thickness (tangential): \(t_{vane} = (1-\mathrm{fill})\cdot p_t\)
      • Cavity depth (Kroll quarter-wave corrected): \(d \approx 0.30\,\lambda\)
      • Rectangular cavity volume estimate: \(V_c \approx d\,w_{gap}\,L_a\)
      • Rectangular cavity surface area estimate: \(A_c \approx 2\,(d\,w_{gap} + d\,L_a + w_{gap}\,L_a)\)
    • Cathode current density
      • Cathode return fraction (McDowell): \(\alpha\approx 0.75\Rightarrow I_c = I_a/(1-\alpha)\)
      • Emission current density: \(J_c = \frac{I_c}{2\pi r_c L_a}\) (reported in A/cm\(^2\))
    • Cavity Q (unloaded) (Kroll/Collins; used as \(Q_U\))
      • \(Q_U \approx \frac{2\pi f\,\mu_0\,V_c}{R_s\,A_c}\)
      • Coaxial type correction used in code: \(Q_U \leftarrow 5.5\,Q_U\)
    • Vane-tip temperature estimate (Carter Eq. 15.155–15.156)
      • Peak anode dissipation: \(P_{a,peak} \approx P_{dc} - P_{rf}\)
      • Average anode dissipation (duty-cycle corrected): \(P_{a,avg} \approx P_{a,peak}\,D\)
      • Heat-flow area: \(A_v = N_v\,t_{vane}\,L_a\)
      • Vane-tip temperature: \(T_v = T_{cool} + \frac{P_{a,avg}\,L_{th}}{A_v\,\kappa_{Cu}}\)
    • UI sensitivity perturbations
      • For each input \(x\in\{f, P, Z_{dc}, \eta, \eta_C\}\), compute \(x_\pm = x\,(1\pm\delta)\) with \(\delta=0.01\), rerun the model, and extract the sweep row at the same \(N_v\).
      • Cells report the perturbed output value and the relative deviation vs base: \(\Delta\% = 100\,(y_\pm - y_0)/y_0\).
    • UI build plausibility (teaching heuristics)
      • Total DC input power needed: \(P_{dc} = P_{rf}/\eta\).
      • Anode loss (peak): \(P_{loss,pk}=P_{dc}-P_{rf}\).
      • Anode loss (average): \(P_{loss,avg}=D\,(P_{dc}-P_{rf})\).
      • Permanent-magnet plausibility uses a simple threshold on \(B_z\) only (does not design the magnetic circuit).
    • \(\pi-1\) mode separation check (Carter Eq. 15.40 pattern; implemented as an \(n\to n-1\) threshold)
      • \(\omega_{s,n-1} = 2\pi f /(n-1)\)
      • \(B_{0,n-1} = \frac{2\,\omega_{s,n-1}}{(e/m)\,(1-\xi^2)}\)
      • \(r_{n-1} = B_z/B_{0,n-1}\)
      • \(V_{0,n-1} = \frac{r_a^2\,\omega_{s,n-1}^2}{2\,(e/m)}\)
      • Threshold voltage: \(V_{T,n-1} = V_{0,n-1}\,(2r_{n-1}-1)\)
      • Mode separation reported: \(\mathrm{sep}\% = 100\,\frac{V_{T,n-1}-V_a}{V_a}\)
    • DC operating diagram curves (used in the UI plot)
      • Hull cut-off curve (cylindrical form): \(V_H(B) = V_0\,(B/B_0)^2\)
      • Mode threshold line (generalized): \(V_{T,m}(B) = V_{0,m}\,(2\,(B/B_{0,m})-1)\)
    • Kilpatrick RF breakdown heuristic (used for sweep warnings)
      • Kilpatrick relation (classic RF guideline): \(f_{MHz} = 1.64\,E^2\,e^{-8.5/E}\), with \(E\) in MV/m
      • Threshold field \(E_{kp}(f)\) is obtained by solving the above relation for \(E\) at the operating frequency
      • Anode–cathode peak field estimate (coaxial approximation at \(r=r_c\)): \(E_{ak,peak} \approx \frac{V_a}{r_c\,\ln(r_a/r_c)}\)
      • Vane-gap field estimate (slot heuristic): \(E_{vane} \approx \frac{V_0}{w_{gap}}\)
      • Warning logic used in the sweep: warn if \(E/E_{kp} > 1.0\); fatal if \(E/E_{kp} > 1.2\)
    • Frequency stability heuristics (used in the UI comparison table)
      • Pulling (MHz): \(\Delta f_{pull} \approx 0.417\,\frac{f_{MHz}}{Q_L}\)
      • Pushing / electron loading (MHz): \(\Delta f_{push} \approx 0.6\,\frac{f_{MHz}}{Q_L}\)

    Symbols match the API payload: \(V_a, I_a, V_0, V_H, B_0, B_z, r_a, r_c, N_v, L_a\). Constants \((e/m), \varepsilon_0, \mu_0, c, \rho_{Cu}\) are defined in magnetron_design.py.

    Collins / Clogston formulas used
    • Reduced (dimensionless) parameters (Collins Eq. 25)
      • \(b = B/\bar{B}\)
      • \(v = V/\bar{V}\)
      • \(i = I/\bar{I}\)
      • \(g = G_L/\bar{G}\)
      • \(p = P/\bar{P}\)
    • Characteristic scales (Collins Eq. 26)
      • \(\bar{B} = \frac{2\,\omega_s}{(e/m)\,(1-(r_c/r_a)^2)}\)
      • \(\bar{V} = \frac{r_a^2\,\omega_s^2}{2\,(e/m)}\)
      • \(\bar{I} = \frac{2\pi\,a_1(r_a/r_c)}{\left(1-(r_c/r_a)^2\right)^2\,(r_a/r_c+1)}\,\frac{m}{e}\,\omega_s^3\,r_a^2\,\varepsilon_0\,L_a\)
        Here \(a_1(r_a/r_c)\) comes from Collins Fig. 10‑9 (digitized and embedded as a piecewise‑linear curve).
      • \(\bar{G} = \bar{I}/\bar{V}\)
      • \(\bar{P} = \bar{I}\,\bar{V}\)
    • Intermediate definitions used in this tool
      • Operating mode angular frequency: \(\omega_s = 2\pi f / (N_v/2)\)
      • In code, \(p\) uses \(P =\) requested RF output power (so it reflects the operating point and efficiency assumptions).
      • \(g\) uses an estimated load conductance \(G_L\) (so \(g\) is approximate):
        • Equivalent capacitance (coaxial approximation): \(C_{eq} \approx 2\pi\varepsilon_0 L_a/\ln(r_a/r_c)\)
        • External \(Q\) estimate: \(1/Q_L = 1/Q_U + 1/Q_{ext}\Rightarrow Q_{ext}=1/(1/Q_L-1/Q_U)\)
        • Conductance from \(Q\): \(G_L \approx \omega_s C_{eq} / Q_{ext}\) (fallback: \(\omega_s C_{eq} / Q_L\))

    • Collins, G.B. et al. (1948). Microwave Magnetrons, MIT Radiation Laboratory Series Vol. 6. McGraw-Hill. — Strap admittance derivation (§5), cavity Q from the Kroll formula (§3), and the Clogston universal dimensionless design parameters (§16).
